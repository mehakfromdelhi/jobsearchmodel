# Website App

This folder contains the browser-native product scaffold for Mehak's Job Search Model.

## Included

- landing page
- magic-link sign-in screen
- multi-step onboarding UI
- dashboard home
- job detail page
- resume manager
- settings and integrations page
- Prisma schema for the future multi-user data model

## Not fully wired yet

- real email delivery for magic links
- real auth sessions
- persistent database reads and writes
- production scan orchestration
- OAuth for Google Sheets and Notion

## Run locally

```bash
cd website
npm install
npm run dev
```
